# Генератор фракталов

Простая программа на C, которая рисует фракталы и сохраняет их в файлы формата PGM.

Рисует три вида:
1. Множество Мандельброта
2. Множество Жюлиа
3. Burning Ship (Горящий корабль)

## Как запустить

Вам понадобятся `gcc` и `make`.

1. **Сборка:**
   ```bash
   make
   ```

2. **Запуск:**
   ```bash
   ./fractal_gen
   ```

После запуска в папке появятся три файла: `mandelbrot.pgm`, `julia.pgm`, `ship.pgm`.
